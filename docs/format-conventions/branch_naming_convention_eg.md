### Template

Plaintext

```
issue-<issue number>-<name>-<CA layer/test>-<use case/module name, separated by hyphens>
```

_(Note: CA refers to Clean Architecture)_

### Examples

- `issue-10-GuanchengChen-UserCase-SwipeRightUseCase`
    
- `issue-11-GuanchengChen-Test-UserCaseInteractorTest`
    
- `issue-12-GuanchengChen-Entity-ResearcherProfile`
